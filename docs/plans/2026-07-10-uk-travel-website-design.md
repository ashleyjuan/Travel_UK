# 英國旅遊網站設計文件

日期：2026-07-10

## 目標

製作一個「旅遊書」式靜態網站，規劃 2026 年 9 月中台北出發、9-10 天的英國（倫敦為主）自由行。部署到 GitHub Pages，手機優先、中文介面。

## 旅行參數

- 出發：台北（TPE），轉機可接受（看價格）
- 日期：2026/9/7 出發 → 9/16 返台（含飛行 10 天，英國 8 個完整天）
- 範圍：倫敦為基地 + 3 個近郊一日遊（不搬行李、不租車、全大眾運輸）
- 興趣：歷史古蹟與博物館、哈利波特影視場景、美食市集購物、鄉村自然風光
- 預算：盡量精簡但難得體驗不手軟；行程充實中帶悠閒

## 行程骨架

| 天數 | 安排 |
|------|------|
| D1 | 台北 → 倫敦（轉機），傍晚抵達、入住 |
| D2 | 倫敦經典：西敏寺、大笨鐘、白金漢宮衛兵交接、國家美術館 |
| D3 | 倫敦歷史：倫敦塔、塔橋、Borough Market、聖保羅大教堂 |
| D4 | 一日遊①：牛津 + Bicester 折扣村 |
| D5 | 大英博物館、柯芬園、王十字 9¾ 月台、音樂劇 |
| D6 | 一日遊②：科茲窩（水上柏頓、拜柏里）／備選：七姊妹白崖 |
| D7 | 一日遊③：華納兄弟哈利波特影城 + 諾丁丘 |
| D8 | 肯辛頓宮、自然史博物館、下午茶、牛津街購物 |
| D9 | 格林威治或市集自由活動 → 機場 |
| D10 | 抵達台北 |

每日行程需含：**雨天備案（☀️/🌧️ 切換）**、部分天數提供 A/B 備選方案。

## 網站結構（單頁式長捲動 + 頂部導覽）

1. **總覽** — 航班資訊、天氣概況、預算摘要、即時小工具
2. **行前準備** — 機票價格參考、必訂清單（影城票、音樂劇、火車票）、eSIM/簽證（ETA）/退稅
3. **每日行程 D1-D10** — 時間軸卡片：景點（開放時間/門票/停留時間）、Google Maps 導航按鈕、雨天備案切換、交通指引
4. **交通攻略** — Oyster/感應支付、Trainline 購票、機場進市區比較
5. **美食清單** — 依區域整理、附地圖連結

## 技術決策

- 純 HTML/CSS/JS 靜態網站，無框架無建置，行程內容存 JSON
- Google Maps 導航連結格式：`https://www.google.com/maps/dir/?api=1&destination=<lat,lng或名稱>`
- 即時小工具：Open-Meteo（倫敦天氣）、免費匯率 API（GBP↔TWD）；離線時優雅降級隱藏
- 部署：GitHub Pages

## 製作順序

1. 網路搜集資料（航班價格、景點票價與開放時間、交通）
2. 整理成行程 JSON
3. 建網站（版面 → 每日卡片 → 地圖連結 → 小工具）
4. 部署 GitHub Pages
